使用Using关键字有效回收资源

Using对象必须实现IDisposable接口或则其基类实现了IDisposable接口，此接口主要用途是释放非托管资源。
同时也可以使用多个Using关键字。

/// <summary>
/// Program类
/// </summary>
class Program
{
    /// <summary>
    /// 程序入口
    /// </summary>
    /// <param name="args">参数数组</param>
    static void Main(string[] args)
    {
        using (A a = new A())
        using (B b = new B())
        {
            //对象a和对象b操作
        }
        Console.ReadKey();
    }
}

判断年份是否是闰年

普通闰年:能被4整除但不能被100整除的年份为普通闰年。（如2004年就是闰年，1999年不是闰年）；
世纪闰年:能被400整除的为世纪闰年。（如2000年是闰年，1900年不是闰年）。
